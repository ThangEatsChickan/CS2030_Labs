Burger

Topic Coverage

  • Immutability
  • Method chaining
  • List
  • Stream (optional)

Problem Description

Burgers are basically edible stacks (change my mind). In this exercise we will create burgers with ingredients that can be stacked as high as your heart desires.
With a bit of imagination, a burger can be represented in the following form:

bun
sauce
lettuce
cheese
patty
bun

Your Burger will have the following simple methods:

  1. Adding ingredient
  2. Removing ingredient
  3. Completing the burger

Do keep in mind that we will be implementing it with immutability and method chaining.

Level 1

Define the Burger class with a static method used to create the burger, and a method to complete the burger. For simplicity, we will be using Strings to represent the ingredients. A burger always begins with a bun, and ends with a bun.

$ javac Burger.java
$ jshell Burger.java
jshell> Burger burger = Burger.of("patty")
$.. ==>
patty
bun

jshell> burger.done()
$.. ==>
bun
patty
bun

jshell> burger.done().done()
$.. ==>
bun
patty
bun

jshell> Burger.of("fillet").done()
$.. ==>
bun
fillet
bun

jshell> /exit

Level 2

Now we can create a method to stack ingredients in the burger. Think of it as adding the ingredients layer by layer from bottom up. Important to note that when a burger is completed, you cannot stack any more ingredients. 

$ javac Burger.java
$ jshell Burger.java
jshell> Burger.of("patty").and("cheese").done()
$.. ==>
bun
cheese
patty
bun

jshell> Burger.of("bacon").and("patty").and("lettuce").done().and("egg")
$.. ==>
bun
lettuce
patty
bacon
bun

jshell> Burger b1 = Burger.of("patty")
jshell> Burger b2 = b1.done().and("pineapple").and("pineapple")
$.. ==>
bun
patty
bun

jshell> Burger b3 = b1.and("pineapple").done()
$.. ==>
bun
pineapple
patty
bun

jshell> /exit

Level 3

Finally, we can implement a way to remove ingredients in case we add regrettables onto the burger! Sadly, once the burger is done, we cannot remove any ingredient.

$ javac Burger.java
$ jshell Burger.java
jshell> Burger.of("bacon").and("bacon").and("bacon").remove("bacon").done()
$.. ==>
bun
bacon
bacon
bun

jshell> Burger.of("patty").and("typo").remove("typo").and("cheese").done()
$.. ==>
bun
cheese
patty
bun

jshell> Burger.of("patty").and("pineapple").and("lettuce").remove("pineapple").done()
$.. ==>
bun
lettuce
patty
bun

jshell> Burger.of("fillet").and("tartar sauce").done().remove("fillet")
$.. ==>
bun
tartar sauce
fillet
bun

jshell> /exit
