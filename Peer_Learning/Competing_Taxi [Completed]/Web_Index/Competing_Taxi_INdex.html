Problem Description

PREREQUISITE: Please complete the pre-requisite task "Competing Taxis" first: here

Now that you are familiar with the process of creating simple Taxi objects, let us now extend this so that we are now able to create immutable Taxi objects.

If the design in your original program was right, this should be achievable with only a few lines of new code, or modifications to your existing solution.

Remember to use the final keyword to highlight your intent, where applicable.

Modify your original code such that the original code now returns new Taxi objects, instead of modifying the original Taxi object.

Level 1

Update your Taxi class object so that the original Taxi object is now immutable.

$ jshell -q your_files_in_ascending_dependency_order
jshell> new Taxi("Grab").travel(5.0)
$.. ==> Grab with total fare $10.70
jshell> new Taxi("Go-Jek").travel(52.6)
$.. ==> Go-Jek with total fare $52.34
jshell> /exit

Level 2

Update your Main class to take advantage of this new immutability.

The following is a sample run of the program. User input is underlined.

$ java Main
Grab
50.4
Grab with total fare $78.80
