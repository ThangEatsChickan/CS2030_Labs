Problem Description

You are given a task to design a food delivery system for a food delivery company to keep track of the delivery man in each location.  The food delivery company consist of two type of delivery man: a normal delivery man (Deliver food on foot) and Rider (Deliver food using vehicle).

 

Delivery Man

All delivery man has two attributes: a id, name.

Rider

Rider is a Delivery Man who delivery food using their own vehicle. Rider has three attributes: a id, name and licenceDate.

Food Delivery

The food delivery company have a list of delivery man operate in certain location. It has two attributes: location and list of delivery man.

 

 

Level 1

 

Create a DeliveryMan class

 

Design an immutable class DeliveryMan that take in a unique id as int, and the name as string

You are also required to implement a changeName(Name) method which will update the name of the rider.

 

jshell> /open DeliveryMan.java    

jshell> DeliveryMan dm1 = new DeliveryMan(1,"Alice")                                                                   

 ==> Delivery Man 1 :  Alice                                                                                                                                                                 

jshell> dm1=dm1.changeName("Jerry")                                                                                    

==> Delivery Man 1 :  Jerry                                                                                                                                                                                                                 

jshell> dm1                                                                                                             

 ==> Delivery Man 1 :  Jerry                                                                                        

jshell> /exit 

 

 

 

Level 2

Create a Rider class

 

Design an immutable class Rider that take in in a unique id as int, name as string, and licenceDate as String.

 

You are also required to implement a changeName(Name) method which will update the name of the rider.

 

You are also required to implement a changeDate(Date) method which will update the date of the licence.

 

jshell> /open DeliveryMan.java

jshell> /open Rider.java

jshell> Rider r1 = new Rider(1,"Mary","201212");

 ==> Rider 1 : Mary (201212)

 

jshell> r1=r1.changeName("Jame");

 ==> Rider 1 : Jame (201212)

 

jshell> r1=r1.changeDate("201219");

 ==> Rider 1 : Jame (201219)

 

jshell> r1

==> Rider 1 : Jame (201219)

jshell> /exit 

 

Level 3

 

Create a FoodDelivery class

 

 Create a FoodDelivery class that take in a list of deliveryman and location as String.

jshell> /open Rider.java

jshell> /open FoodDelivery.java

Level 4

 

Create a Main class

 

Create a main class that read in

  • a int which represent the number of location.
  • For each location read in
    • location name 
    • Number of riders in the location 
    •  Type of Delivery man represent using a single character(D/R) 
      • D -> Normal DeliveryMan
      • R -> Rider
  • DeliveryMan Info (name [and licenceDate])
