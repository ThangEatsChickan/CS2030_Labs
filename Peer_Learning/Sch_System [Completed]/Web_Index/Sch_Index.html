CS2030 Peer Learning: School System

Topic Coverage

  • Object-Oriented Design
  • Inheritance

Project Description

You are given a task to design a school system that allows computation test and exam marks of the students and get the student with highest total marks for both test and exam.

Person

All persons have two attributes: nric and name.

Student

Student is a person that has matriculation number, test mark as well as exam mark.

Level 1

Create a Person class

Define an immutable Person class that takes in an unique nric as string and the name as string.Implement the view() method to view the information of the person.

$ jshell Person.java
jshell> Person p = new Person("S1234567A", "Elicia")
jshell> p.view()
Nric: S1234567A
Name: Elicia
jshell> /exit

Level 2 

Create a Student class

Define a Student class. Its arguments of the constructor are his nric, name, matriculation number, test mark as well as exam mark.

Implement a method: computeFinalMark() that returns the computation of total mark of test and exam marks. (test mark + exam mark /2)

jshell> /open Student.java
jshell> Student stud = new Student("S2345678B","Michelle", "A1212345A", 50, 70)
jshell> stud.computeFinalMark()
jshell> stud.view()

Nric: S2345678B
Name: Michelle
Admin No: A1212345A
Test Mark: 50.0
Exam Mark: 70.0
Final Mark: 60.0
jshell> /exit

Level 3

Create a Main class

Create Main class that reads in

  • number of students
  • nric, name, matriculation number, test and exam mark of each student

You are to implement a static method: topStud that takes in the a list of students inputs and returns the student with highest computation of test and exam marks.

User input is underlined.

$ javac Main.java
2
S1234A Elicia 1234A 12 50
S1234B Michelle 123B 70 80
Nric: S1234B
Name: Michelle
Admin No: 123B
Test Mark: 70.0
Exam Mark: 80.0
Final Mark: 75.0
